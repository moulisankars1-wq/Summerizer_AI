# 🧠 Text Summarizer AI

## 📌 Project Overview
This project is a simple Text Summarization Tool developed using Python.
It generates a concise summary from a given paragraph using a
frequency-based extractive summarization technique.

The system analyzes the input text, calculates the frequency of words,
scores each sentence based on word importance, and selects the most
relevant sentences to produce a meaningful summary.

---

## 🎯 Objectives
- To reduce long paragraphs into short summaries
- To identify important sentences using word frequency
- To implement text summarization using pure Python logic
- To develop an NLP-based application without external libraries

---

## ⚙️ Features
- Extractive Text Summarization
- Frequency-Based Algorithm
- No External NLP Libraries Used
- Lightweight Implementation
- Mobile (Termux) Compatible
- Generates Summary from User Input

---

## 🛠️ Technologies Used
- Python
- Vim Editor
- Termux
- Git & GitHub

---

## ▶️ How to Run the Project

1. Open Termux
2. Navigate to the project folder
3. Run the following command:

   python summarizer.py

4. Enter a paragraph when prompted
5. The summarized output will be displayed
